FAQ

よくあるご質問

気になることは何でもカウンセリングでお聞きいただけます。

Q 運動が久しぶりで体力に自信がありません。大丈夫ですか？ +
まったく問題ありません。FitCoreにいらっしゃる方のほとんどが、 長期間運動から離れていた方です。カウンセリングで現在の体力・健康状態を 丁寧に確認したうえで、無理のないところからスタートします。
Q 無料カウンセリングで、その場で入会を勧められますか？ +
一切ありません。カウンセリングはあなたの話を聞き、 何が合っているかを一緒に考える場です。 プランのご提案はしますが、その日に決断を求めることはなく、 じっくり検討していただいてから入会いただけます。
Q 忙しくて毎週通えるか不安です。 +
週1回からのプランも用意しています。また、お仕事が立て込む時期には スケジュール調整も柔軟に対応可能です。 続けられる無理のないペースを一緒に設計しますので、 まずはカウンセリングでご状況をお聞かせください。
Q 食事制限は厳しいですか？ +
「食べないダイエット」は行いません。管理栄養士の資格を持つトレーナーが、 今の食生活をベースに「食べながら変わる」プランを提案します。 極端な制限はリバウンドの原因になるため、長く続く方法を重視しています。
Q オンラインでのサポートはありますか？ +
はい。LINEを通じたセッション外サポートに加え、 オンライントレーニングセッションにも対応しています。 出張が多い方や、都合に合わせて使い分けたい方にも好評です。
Q 途中でやめることはできますか？縛りはありますか？ +
最低契約期間は3ヶ月ですが、その後は1ヶ月単位で更新・停止が可能です。 急な転勤・体調変化などのやむを得ない事情がある場合は、 ご相談のうえ柔軟に対応いたします。強引な引き止めは一切行いません。
Q 産後や不規則な仕事をしていても通えますか？ +
はい、対応しています。産後の方には身体の回復状況に合わせた 段階的なプログラムをご用意しています。シフト勤務・夜勤の方には 早朝（7時〜）・夜間（21時まで）の時間帯も対応しており、 スケジュール変更も柔軟に承ります。
